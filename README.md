## 📊 Dataset Overview

The dataset includes environmental and weather conditions:

| Feature            | Description                   |
| ------------------ | ----------------------------- |
| Temperature        | Atmospheric temperature       |
| RH                 | Relative Humidity             |
| Ws                 | Wind Speed                    |
| Rain               | Rainfall                      |
| FFMC, DMC, DC, ISI | Fire Weather Index components |
| Region             | Bejaia / Sidi Bel-Abbes       |
